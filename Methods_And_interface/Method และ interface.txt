ใน Go method และ interface เป็นส่วนสำคัญที่ช่วยให้การออกแบบและเขียนโปรแกรม
เป็นไปอย่างยืดหยุ่นและมีประสิทธิภาพ 
โดยเฉพาะเมื่อจัดการกับการออกแบบโครงสร้างและพฤติกรรมของชนิดข้อมูลต่างๆ
1.Method
- Method คือ ฟังก์ชันที่มี receiver ซึ่งเป็นชนิดข้อมูลที่กำหนดไว้และสามารถใช้เพื่อ
ดำเนินการกับข้อมูลของชนิดนั้นๆ
- Method สามารถกำหนดบนชนิดข้อมูลที่เป็น struct หรือ type อื่นๆ ได้
2.interface
อธิบาย 
Interface คือ ชนิดข้อมูลที่กำหนดชุดของ method signatures (ชื่อ method และพารามิเตอร์)
ชนิดข้อมูลอื่นๆ ต้อง implemet
Interface ช่วยให้เราสามารถกำหนดพฤติกรรมที่คาดหวังจากชนิดข้อมูลต่างๆ ได้ 
โดยไม่ต้องรู้รายละเอียดการทำงานภายใน
