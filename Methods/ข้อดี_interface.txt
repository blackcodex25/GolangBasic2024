ข้อดีของ interfaces
1.ยืดหยุ่น: interface ช่วยให้เราสามารถเขียนฟังก์ชันที่ทำงานกับชนิด
ข้อมูลที่ implement method ที่กำหนดใน interface นั้นๆ 
โดยไม่ต้องรายละเอียดการทำงานภายในของชนิดข้อมูล
2.การใช้ Polymorphism: เราสามารถใช้ interface 
เพื่อจัดการกับชนิดข้อมูลที่มีพฤติกรรมคล้ายคลึงกัน แม้ว่า
implementation อาจแตกต่างกัน
3.การทดสอบ: interface ทำให้การทดสอบง่ายขึ้นเพราะเราสามารถสร้าง mocks 
หรือ stubs ที่ implement interface สำหรับการทดสอบได้
