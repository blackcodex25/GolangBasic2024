การใช้ switch ในภาษา Go
คำสั่ง switch ในภาษา Go เป็นวิธีที่ย่อให้สั้นลงสำหรับการเขียนลำดับของคำสั่ง if - else
โดยมันจะทำงานกับ case แรกที่มีค่าเท่ากับค่าเงื่อนไข คำสั่ง switch ในภาษา Go 
มีลักษณะคล้ายกับ switch ในภาษา C, C++, JavaScript, และ PHP แต่มีความ
แตกต่างบางประการที่สำคัญ
1.ภาษา Go จะทำงานเฉพาะ case ที่ถูกเลือกเท่านั้น ไม่ใช่ทุก case ที่ตามมา ซึ่งทำให้ไม่จำเป็น
ต้องใช้คำสั่ง break ในแต่ละ case 
2.case ใน Go ไม่จำเป็นต้องเป็นค่าคงที่ (constants) 
และค่าใน case ไม่จำเป็นต้องเป็นจำนวนเต็ม (integers)
