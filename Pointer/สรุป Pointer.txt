สรุป
สำหรับ Go การกำหนดค่าให้ตัวแปร 
หรือการส่งค่าและรับค่าจาก function ทุกอย่างคือการ Copy ค่าเสมอ
ส่วนถ้าต้องการแชร์ข้อมูลข้ามขอบเขตของ function 
จะใช้ Pointer type ช่วยเพราะมี Operator เครื่องหมายสตาร์ * 
ในการเข้าถึงหรือแก้ไขข้อมูลในตำแหน่งที่ pointer เก็บเอาไว้อยู่

ประโยชน์ของ Pointer
1.ประหยัดหน่วยความจำ: การใช้ pointer ลดการคัดลอก ข้อมูลขนาดใหญ่
ทำให้โปรแกรมทำงานมีประสิทธิภาพมากขึ้น
2.การแชร์ข้อมูล: ใช้ Pointer เพื่อแชร์และแก้ไขข้อมูลระหว่างฟังก์ชันต่างๆ
3.การใช้ Pointer กับ struct, slice, map: การเข้าถึงและแก้ไข
ข้อมูลใน struct, slice และ map ผ่าน Pointer
